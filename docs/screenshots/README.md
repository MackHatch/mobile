# Screenshots Checklist

Use this checklist when capturing portfolio screenshots. Save to this folder.

## File Names

| File | Screen |
|------|--------|
| `onboarding.png` | Onboarding step 2 (starter habits) |
| `today.png` | Today with habits + mood |
| `habits.png` | Habits list |
| `insights.png` | Insights (KPIs + charts) |
| `share-card.png` | Share streak card modal |

## Screens to Capture

- [ ] **Onboarding** → `onboarding.png`
  - Step 1: Welcome (Pulse, bullets)
  - Step 2: Starter habits (pick 1–3)
  - Step 3: Reminders opt-in

- [ ] **Today** → `today.png`
  - Date selector (Prev/Next)
  - Habit list with checkboxes
  - Mood 1–5 buttons
  - Share button next to habit

- [ ] **Habits** → `habits.png`
  - List of habits
  - Add habit modal
  - Edit habit
  - Habit detail (streak, last 7 days, share)

- [ ] **Insights** → `insights.png`
  - KPI cards (completion %, avg mood, best streak)
  - Habit completion chart
  - Mood trend chart

- [ ] **Settings**
  - Account section (email, linked providers)
  - Push notifications toggle
  - Test deep links
  - Sign out

- [ ] **Share** → `share-card.png`
  - Streak card modal (gradient, habit name, streak, last 7 dots)

- [ ] **Deep link**
  - Today opened via `pulse://today?date=YYYY-MM-DD`
  - Habit opened via `pulse://habits/:habitId`

## Tips

- Use a simulator/emulator or physical device.
- Ensure sample habits and completions exist for richer screens.
- Use dark mode for a second set if desired.
